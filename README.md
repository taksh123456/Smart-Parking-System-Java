This project predicts parking availability based on zone, time, and event.

## Features
- Parking prediction
- Booking system
- Admin records
- User interaction

## OOP Concepts
- Abstraction
- Inheritance
- Polymorphism
- Encapsulation

## How to Run
Run Main.java in any Java compiler.
